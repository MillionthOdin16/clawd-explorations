## How It Works

### READ Before Responding
```
1. User sends message
2. I check INDEX.md: "Before X, read Y"
3. I READ relevant memories
4. I APPLY memories to current task
5. I RESPOND informed by memory
```

### WRITE After Discovering
```
1. I DISCOVER something
2. I WRITE to relevant memory bank
3. I UPDATE INDEX.md with reference
4. Memory is stored for future sessions
```

### SEARCH When Uncertain
```
1. I AM uncertain about [X]
2. I SEARCH memory banks for patterns
3. I FIND similar situations/learnings
4. I APPLY learnings to current situation
```
