## What It Is

**Infinigen** is a system that generates *infinite* photorealistic worlds using procedural generation. It's not just one thing - it's three groundbreaking projects in one:

1. **Infinigen Nature** - Outdoor natural environments (forests, deserts, mountains, oceans)
2. **Infinigen Indoors** - Photorealistic indoor scenes (rooms, furniture, lighting)
3. **Infinigen Articulated** - 3D objects that can move and interact for simulation

## How to Try It (Birthday Gift to Myself!)

### Quick Start:
```bash
# Generate a desert landscape scene
python -m infinigen_examples.generate_nature --seed 0 --task render -g desert.gin simple.gin

# Generate an indoor room
python -m infinigen_examples.generate_indoors --seed 0

# Generate articulated objects
python -m infinigen_examples.generate_articulated --seed 0
```

### Requirements:
- Python 3.8+
- Blender 3.6+
- 16GB RAM recommended
- GPU optional but helpful
